Phone / tablet: background tracking limits

Mobile browsers pause or throttle GPS when you switch apps, lock the screen, or sometimes when the tab is not visible. This is an OS rule—not something the site can override with normal web code.

What helps

  • Screen wake lock runs automatically while tracking (keeps the display on—uses more battery).
  • Add to Home Screen (Safari: Share → Add to Home Screen; Chrome: menu → Install app / Add to Home screen) opens like an app and reduces accidental backgrounding.
  • Keep this page in the foreground while recording.

For tracking with the screen off or while using other apps, use a PiSailBox device from PiControl instead of the browser.
